# ProjectEuler
Repository with all my algo solutions for the exercises of the Project Euler website.

## How to run
To run the code use the command below (Change the [FOLDER_NAME] with the name of the folder of the exercise you want to run):
```bash
go run ./[FOLDER_NAME]/main.go
```

### Example
```bash
go run ./MultiplesOf3and5/main.go
```
